My Project Idea

Overview

I am planning to build a website that allows users to learn the objective personality system (OPS) (Personality, 2022). OPS is based on MBTI, the theory proposed by Carl Young. The website would provide a brief overview of the system then introduce the user to chart where they can experiment with system; for example, if they selected that they were an extrovert it removes all the introverted types from the chart with a brief explanation why the types were removed. It would aim to help you learn OPS without having to go through mounds of content. This would help kept the process fun and engaging for the user.

motivation

This website would be useful introduce people to OPS and allow people understand what the types mean. If you are not familiar with OPS or MBTI it can be almost impossible to understand as those already familiar use a lot of jargon and acronyms. This tool would help people to understand the terminology and learn to type. The only thing like this is subjectivepersonality.com (Hosler, 2020) but it requires reading a lot of text and the typing tool requires you to already have an understanding of the system to be useable.

Description

OPS has 9 binary coins, 8 functions, 4 human needs, and 4 animals. All these stacks and interact with one another, which can lead to much confusion. The introduction page would include a sentence description of each part followed by the typing chart. The typing chart is the focus as it is the unique part of the project.

Part One the Chart

This would look like Hosler’s typing chart prompting the user to input each data point individually, and it would eliminate the types excluded by the input, i.e. inputting that the user is a Senser would remove all Intuitive types. Every time the user made a change a small icon would appear which if selected would explain why types were eliminated or made available again (Hosler, 2020).

Part Two Learning to Type

The other aspect of this project is helping users to learn to type. To achieve this the website would allow users to type a person about whom information have been uploaded to the site to aid in typing them. For example, interview clips of celebrities could be linked on the site, allowing multiple users to review the clips and post their analysis of which type would best describe the person. This analysis could then be shared with other users of the website. Other users can provide feedback on the analysis performed and share their own analysis.

Tools and Technologies

Investigation would need to be undertaken to determine the best technological options to host the website, store the information, secure the account information for users and to perform user authentication.

Skills Needed

To make this project happen I would need to learn front-end languages to build the user interface like HTML and CSS with JavaScript so the users can interact with the website. I need to learn how to authenticate users and how to manage any privacy or legal issues regarding the videos being linked to. I would need to learn python to use for parts of the backend because it’s easier to learn (Giampedraglia, 2019). In addition, a basic understanding of good user experience design and graphic design would be need. If the data is being collected an understanding of how to store and interpret the data to improve the performance would be beneficial. To launch the website, I would need to learn how to host and maintain a website.

Outcome

If the project works, there would be fun and interactive way understand OPS and with simple easy to use tool to learn objective personality system, learn to type other people and allow people to be typed by others in the community. If the project was highly successful it would help increase the popularity of the system and enable it to be tested in more through and scientific fashion by analyzing the data produced from the website. It would provide way of understanding your own type without having to learn the entire system (which has 512 types in total).
